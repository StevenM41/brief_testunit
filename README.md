# 🎯 Voir les tests unitaires

Une application Laravel basée sur PHP, conçue pour la réalistion des tests unitaires.

---

## 🚀 Prérequis

- PHP >= 8.2
- Composer
- Laravel >= 10

---

## ⚙️ Installation

1. **Cloner le dépôt**
   ```bash
   git clone https://github.com/StevenM41/brief_testunit.git
   cd test_unitaire
   ```
